Alternative collider bias corrections

If you want to correct the original subsequent gwas (/user/work/wt23152/test_data/data/gwas/subsequent_standardised.tsv.gz) using different BETA and SE as calculated in collider bias results section above, please run this command inside a sbatch script

module add apps/singularity/3.8.3
singularity run -B /user/work/$(whoami) -B /user/home/$(whoami) --pwd /home/r_scripts/ \
    docker://andrewrrelmore/genepi_pipeline:latest \
    adjust_gwas_for_collider_bias.r --gwas `SUBSEQUENT_GWAS` \
        --harmonised_effects_gwas `HARMONISED_GWAS` \
        --beta <BETA> \
        --se <SE> \
        --output_gwas <OUTPUT_FILE_NAME>
